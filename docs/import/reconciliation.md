## Защитные решения

- Формулы не исполняются, cached formula result не используется как доверенное значение.
- СНИЛС и дата рождения редактируются до raw staging: сохраняются координаты,
  заголовок и hash строки, но не исходное значение.
- Явные тестовые/служебные значения, отсутствие ФИО и однозначная бессмыслица дают
  `REJECTED` observation; полностью непригодная строка получает `IGNORED`. Исходная
  строка не удаляется, но участник и его операционные связи не создаются.
- ФИО и контакты не являются уникальным ключом. `AUTO_DEDUPE_V1` объединяет только
  совместимые нормализованные имена при наличии чистого общего контакта и
  отсутствии конфликта external ID в одном namespace.
- Неоднозначные совпадения закрываются как `DISMISSED`, явные конфликты — как
  `NOT_DUPLICATE`; merge выполняется транзакционно, журналируется и обратим.
- Полный набор неотредактированных source fields доступен только ролям с
  `imports.read_raw`; bulk CSV — только ролям с `exports.bulk`.
